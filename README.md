# Zenn CLI

* [📘 How to use](https://zenn.dev/zenn/articles/zenn-cli-guide)

# usage

## 記事の悪性

```bash
npx zenn new:article
npx zenn new:article --slug xxx
```

## 本の作成

```bash
npx zenn new:book
npx zenn new:book --slug xxx
```

## ブラウザでプレビュー

```bash
open http://localhost:8000 && npx zenn preview --port 8000
```
